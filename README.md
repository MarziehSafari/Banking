# Banking
### Banking and financial services  

#### Portfolio Executive Summary

This repository features a dual-framework approach to credit risk, transitioning from static binary classification to dynamic temporal modeling. To maximize bank profitability, I developed a suite of machine learning classifiers—including **XGBoost, Decision Tree, Random Forest, and ExtraTrees**. By optimizing the decision threshold to **0.42**, I successfully balanced interest income against potential principal loss, ultimately driving higher **net profit** for the portfolio. I then expanded this analysis into **Survival Analysis (Kaplan-Meier & Cox Proportional Hazards)** to investigate the **'Time-to-Even'**t. By identifying a critical hazard window **between months 5 and 40** and integrating macroeconomic indicators from FRED, I demonstrated how factors like the **Interest Rate** and **borrower-level traits (Annual income, Home Ownership)** drive the velocity of default. Together, these projects provide a comprehensive toolkit for proactive portfolio intervention.
