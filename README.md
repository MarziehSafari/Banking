# Banking
### Banking and financial services  

#### Portfolio Executive Summary
This repository features a dual-framework approach to credit risk, transitioning from static binary classification to dynamic temporal modeling.  

Initially, I developed a suite of Machine Learning classifiers (XGBoost, Desicion Tree, Random Forest, ExtraTrees) to predict the probability of default, then expanded this analysis into Survival Analysis (Kaplan-Meier & Cox Proportional Hazards) to investigate the 'Time-to-Event.' By identifying a critical hazard window between months 5 and 40 and integrating macroeconomic indicators from FRED, I demonstrated how factors like the Interest Rate and borrower-level traits (Annual income, Home Ownership) drive the velocity of default. Together, these projects provide a comprehensive toolkit for proactive portfolio intervention.
